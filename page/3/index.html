联通 WO-36 光猫改桥接

发表于 03-11-2016 • 分类于 瞎鼓捣
• 字数统计: 109
  1. http://192.168.18.1/logoffaccount.html ，隐藏用户改为启用
  2. 登录工程帐号用户名：fiberhomehg2x0 密码：hg2x0
  3. 下载 http://192.168.18.1/backupsettings.conf 或开启 FTP 下载 ftp://192.168.18.1/fhconf/backpresettings.conf
  4. 搜索 AdminPassword ，base64 解码后用户名为 CUAdmin ，密码为 CUAdmin 。
  5. 打开 http://192.168.18.1/cu.html 登录。
  6. 修改桥接设置，F12 查看拨号密码，记下 VID 还原用。

PS：这个光猫只有 LAN1 是千兆口。
